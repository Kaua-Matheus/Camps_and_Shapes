# Camps and Shapes

**Camps and Shapes** é um jogo top-down feito no Godot em que você controla um slime e precisa derrotar monstros enquanto explora o cenário.

O projeto usa uma estética em pixel art e já inclui sistema de movimento, ataque, dash, barra de vida, pausa, som, menu inicial e inimigos como o golem.

## Objetivo

Sobreviva aos combates, derrote os inimigos e avance pelo mapa com a slime.

## Controles

- `W`, `A`, `S`, `D` ou setas: mover
- Botão esquerdo do mouse: atacar
- `R`: dash

## Como executar

1. Abra o projeto no Godot Engine.
2. Use a cena principal configurada em `project.godot`.
3. Execute o projeto pela interface do editor.

## Estrutura do projeto

- `entities/`: cenas de personagens, inimigos e objetos
- `scenes/`: telas de interface, música de fundo e mapas
- `scripts/`: lógica principal do jogo
- `assets/`: fontes, músicas, sprites e outros recursos

## Observações

O jogo ainda pode estar em desenvolvimento, então algumas cenas e mecânicas podem mudar ao longo do tempo.
